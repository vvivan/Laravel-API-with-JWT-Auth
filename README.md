# Laravel API with JWT Auth

Basic Laravel API with integrated Sean Tymon JWT-Auth Middleware.

## Routes - API Endpoints

### Signup Users
POST Request
/api/user

### Signin Users
POST Request
/api/user/signin

### Get All Notes (Quotes)
GET Request
/api/quotes

### Store the Note (Quote)
POST Request
/api/quote

### Edit the Note (Quote)
PUT Request
/api/quote/{id}
